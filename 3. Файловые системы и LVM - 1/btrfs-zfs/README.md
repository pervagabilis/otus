# Работа с LVM

## Цель
создавать и управлять логическими томами в LVM

## Текст задания

⭐️Задание со звездочкой

На дисках поставить btrfs/zfs — с кэшем, снапшотами и разметить там каталог /opt.

Логировать работу можно с помощью утилиты script.

Формат сдачи: bash-скрипт с настройкой LVM (/, /home, /var mirror, снапшоты, fstab) + краткий отчёт со списком команд для создания/восстановления снапшотов и (при ⭐️) btrfs/zfs.

Задание считается выполненным, если проведены все манипуляции с разделами, прописано монтирование в /etc/fstab, проведена работа со снепшотами.

## Окружение

- Хостовая машина на Windows 11.
- VirtualBox версии 7.2.6 r172322 (Qt6.8.0 on windows)
- Виртуальная машина с Ubuntu Server 24.04.2
  - 2 ядра
  - 2 Гб ОЗУ
  - 10 Гб виртуальный диск
